my_firewall

Description
===========

my_firewall - Testing Puppetlabs firewall 

Use Case
========
 * Turn on firewall
 * enable port 22
 * enable port 80

```
$ bundle install && bundle update && bundle exec rake spec_prep && bundle exec rake spec
$ vagrant up --provision
```

Platform
--------
 * work from CentOS 7
